Forms
Form tag
Syntax
<form>...</form> cointainer for inputs
<form action="/action_page.php"> php file is sent to server on submit
<form target="_blank"> response is displayed in a new window or tab
<form target="_self"> response is displayed in a new window or tab
<form target="_top"> response is displayed in the current window
<form method="get"> appends the form data to the URL, sends to server(not safe)
<form method="post"> safer option
<form autocomplete="on">   
<form novalidate> input should not be validated
<form name="myForm"> name of input for JS
<form accept-charset="UTF-8"> accepted encryption of input
<form enctype="application/x-www-form-urlencoded"> 
Specifies how the form-data should be encoded when submitting it to the server 
(only for method="post")

Input element
Syntax
<input type="text">
<input type="password"> masked characters
<input type="submit"> button for submittinf form data to form-handler(<form action="smt.php">)
<input type="reset"> resets all form values
<input type="radio">
<input type="checkbox">
<input type="button">
<input type="color">
<input type="date">
<input type="email">
<input type="image">
<input type="file">
<input type="hidden">

<input value="John">
<input readonly>
<input disabled>
<input size="50"> max visible characters
<input maxlength="4">
<input multiple> user is allowed to enter more than one value   
<input pattern="[A-za-z]{3}"> defines regular expression for input field
<input placeholder="123-45-678" pattern="[0-9]{3}-[0-9]{2}-[0-9]{3}">   hint for pattern
<input required>
<input autofocus> input field should automatically get focused when the page loads
<input autocomplete="on">

<input type="text" form="form1"> input belongs to "form1"
<input formaction="/action_page2.php"> specifies the URL of the file that will process the form
<input formenctype="multipart/form-data">  how form data should be encoded
<input formmethod="post"> defines the HTTP method for sending form data to URL
