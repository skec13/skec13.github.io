Quotations
Quotation tags
Syntax
<blockquote>...</blockquote> section quoted from another source
<q>...</q> short quotation
<abbr title="description">...</abbr> gives explanation on acronym, like HTML, WWW...
<address>...</address> address format
<cite>...</cite> title of a creative work
<bdo dir="rtl">...</bdo> text right to left
Code

<!DOCTYPE >

        live.
        </blockquote>
