Tables
Table tags
Syntax
<table> defines a table
<th> defines a header cell in a table
<tr> row in table
<td> cell in table
<caption> title of table
<colgroup>
    <col span="2" style="background-color:red">   first two columns are red
    <col style="background-color:yellow">   third column is yellow
</colgroup>
<thead> groups the header content in a table
<tbody> groups the body content in a table
<tfoot> groups the footer content in a table
Code

<!DOCTYPE >
<html lang="en">
    <body>
        <table>
            <colgroup>
                <col span="2" style="background-color: #D6EEEE">
            </colgroup>
            <caption>Monthly savings </caption>
            <tr>
                <th colspan="2">Name</th>
                <th>Person 3</th>
            </tr>
            <tr>
                <td>Emil</td>
                <td>Tobias</td>
                <td>Linus</td>
            </tr>
            <tr>
                <td>16</td>
                <td>14</td>
                <td>10</td>
            </tr>
        </table>
    </body>
</html>
