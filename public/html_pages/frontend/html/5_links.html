Links
Hyperlinks
Syntax

<a href="url">Link text</a> hyperlink
<a target="_self"></a> opens document in the same window/tab
<a target="_blank"></a> opens the document in a new window or tab
<a target="_parent"></a> opens the document in the parent frame
<a target="_top"></a> opens the document in the full body of the window
Code

<!DOCTYPE >
<html lang="en">
    <body>
        <a href="www.google.com" target="_self">Link target _self</a>
        <a href="www.google.com" target="_blank">Link target _blank</a>
        <a href="www.google.com" target="_parent">Link target _parent</a>
        <a href="www.google.com" target="_top">Link target _top</a>
    </body>
</html>
Output
