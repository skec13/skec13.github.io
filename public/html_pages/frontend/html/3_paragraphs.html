Paragraph tags
Syntax
<p>This is a paragraph</p> always starts on a new line
<hr> horizontal rule
<br> line break
<pre>This is a pre element</pre> preformatted text(preserves spaces and line breaks)

<code>x=5</code> computer code
<kbd>Ctrl + S</kbd> keyboard input
<samp>Press F1 to continue</samp> output of a programm
<var>1/2 x b</var> variable
