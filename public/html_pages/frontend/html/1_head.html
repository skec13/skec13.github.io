Head
Head tags
Syntax
<head>...</head> metadata about page

<title>title</title> page title

<link rel="stylesheet href="styles.css"></link> link to css file
<link rel="icon" type="image/x-icon" href="/images/favicon.ico"> favicon

<meta charset="UTF-8"> used charset
<meta name="keywords" content="HTML, CSS, JavaScript"> keywords for search engines
<meta name="description" content="Free Web tutorials"> description
<meta name="author" content="John Doe"> author
<meta http-equiv="refresh" content="30"> refresh rate
<meta name="viewport" content="width=device-width, initial-scale=1.0"> responsiveness

<base href="https://www.w3schools.com/" target="_blank"> base URL

<script src="demo_defer.js" defer></script> 
script downloaded in parallel to parsing the page, and executed after the page has finished parsing
<script async> 
script downloaded in parallel to parsing the page, and executed as soon as it is available
<script crossorigin="anonymous"> 
cross-origin request is performed(no credentials are sent)
<script crossorigin="use-credentials"> 
cross origin request with credentials(cookie, ssh..) 
<script integrity> checks the fetched script for malware
<script type> specifies the media type of the script
<script reffererpolicy="no-refferrer"> no referrer information sent
<script reffererpolicy="origin"> sends(scheme, host, and port)
