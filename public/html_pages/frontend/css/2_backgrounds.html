Backgrounds
Properties
Syntax
div{
    background-color:rgba(0,128,0,0.3);
    background-image:url("image.jpg");
    background-repeat:repeat-x;   /*repeats the background image horizontally*/
                     :no-repeat;
    background-position:right-top;
    background-attachment:fixed;   /*background image will not scrool with the page*/
                         :scroll;
    background-clip:padding-box;   /*background extends behind the border*/
    background-origin:padding-box;   /*images start from the upper left corner of the padding edge*/
                     :border-box;   /*images start from the upper left corner of the border*/
                     :content-box;   /*images start in the upper left corner of the content*/
    background-size:300px 100px;
                   :cover;   /*resizes the background image to cover the entire container*/
                   :contain;   /*resizes the background image to make sure the 
                                 image is fully visible*/
}
