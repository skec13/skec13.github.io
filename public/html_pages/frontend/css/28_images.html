Images
Object-fit
Syntax
img {
    width: 200px;
    height: 300px;
    object-fit: fill;   /*images is resized to fill the given dimension, if necessary it will be stretched or squished*/
                contain;   /*image keeps its aspect ratio, but is resized to fit within the given dimension*/
                cover;   /*image keeps its aspect ratio and fills the given dimension. It will be clipped to fit*/
                none;   /*image is not resized*/
                scale-down;   /*image is scaled down to the smallest version of none or contain*/
}
Code

<!DOCTYPE >
<html lang="en">
<head>
    <style>
        img {
            width: 200px;
            height: 300px;
            border: 1px solid black;
        }
        .first {
            object-fit: fill;
        }
        .second {
            object-fit: contain;
        }
        .third {
            object-fit: cover;
        }
        .fourth {
            object-fit: none;
        }
        .fifth {
            object-fit: scale-down;
        }
        p {
            font-family: Arial;
            font-size: 16px;
        }
    </style>
</head>
<body>
    <p>object-fit: fill;</p>
    <img class="first" src="img.png">
    <p>object-fit: contain;</p>
    <img class="second" src="img.png">
    <p>object-fit: cover;</p>
    <img class="third" src="img.png">
    <p>object-fit: none;</p>
    <img class="fourth" src="img.png">
    <p>object-fit: scale-down;</p>
    <img class="fifth" src="img.png">
</body>
</html>

Object-position
Syntax
img {
    width: 200px;
    height: 300px;
    object-fit: cover;
    object-position: 80% 100%;   /*x y coordinates how image should be positioned*/
}
