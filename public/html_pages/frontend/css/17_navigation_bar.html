Navigation bar
Basic vertical navbar
Syntax
ul {
    list-style-type: none;   /*removes bullets*/
    margin: 0;
    padding: 0;
    width: 60px;  
}

li {
    text-align: center;   /*to center text in li*/
    border-bottom: 1px solid #555;
}

li a {
    display: block;   /*whole link area clickable, can define width*/
}

li a:hover {
    background-color: #555;   /*on hover change element*/
    color: white;
}

.active {
    background-color: #04AA6D;   /*.active class*/
    color: white;
}
