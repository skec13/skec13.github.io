Transforms
2D transforms
Syntax
div {
    transform: translate(x, y);   /*moves element*/
               rotate(angle);   /*rotates element*/
               scale(x, y);   /*scales element*/      
               skew(x-angle, y-angle);   /*disalignes element*/        
               matrix(scaleX(), scaleY(), skewX(), skewY(), translateX(), translateY());     /*matrix of transforms*/        
}
Code

<!DOCTYPE >
<html lang="en">
<head>
    <style>
        .first {
            transform: translate(50px, 50px);
            margin-bottom: 100px;
        }
        .second {
            transform: rotate(20deg);
            margin-bottom: 120px;
        }
        .third {
            transform: scale(0.5, 1);
            margin-bottom: 120px;
        }
        .fourth {
            transform: skew(20deg, 10deg);
            margin-bottom: 100px;
        }
        .fifth {
            transform: matrix(1, -0.3, 0, 1, 0, 0);
        }
    </style>
</head>
<body>
    <div class="default">Default div</div>
    <div class="first">Translated</div>
    <div class="second">Rotated</div>
    <div class="third">Scaled</div>
    <div class="fourth">Skewed</div>
    <div class="fifth">Multiple transforms</div>
</body>
</html>
Output
3D transforms
Syntax
div {
    transform: translate3d(x, y, z);   /*moves element*/
                rotate3d(x, y, z, angle);   /*rotates element*/
                scale3d(x, y, z);   /*scales element*/            
                matrix3d(...);   /*matrix of transforms*/                
}
Code

<!DOCTYPE >
<html lang="en">
<head>
    <style>
        .first {
            transform: rotate3d(1,1,1, 80deg);
        }
    </style>
</head>
<body>
    <div class="default">Default div</div>
    <div class="first">Rotated</div>
</body>
</html>
Output
