Gradients
Linear gradient
Syntax
background-image: linear-gradient(direction, color-stop1, color-stop2, ...);
background-image: linear-gradient(angle, color-stop1, color-stop2);
Code

<!DOCTYPE >
<html lang="en">
<head>
    <style>
        .first {
            background-image: linear-gradient(red, yellow);
        }
        .second {
            background-image: linear-gradient(to right, red , yellow);
        }
        .third {
            background-image: linear-gradient(to bottom right, red, yellow);
        }
        .fourth {
            background-image: linear-gradient(180deg, red, yellow);
        }
        .fifth {
            background-image: linear-gradient(to right, red,orange,yellow,green,blue,indigo,violet);
        }
        .sixth {
            background-image: linear-gradient(to right, rgba(255,0,0,0), rgba(255,0,0,1));
        }
    </style>
</head>
<body>
    <div class="first">Top to bottom</div>
    <div class="second">Left to right</div>
    <div class="third">Top left to bottom right</div>
    <div class="fourth">180deg</div>
    <div class="fifth">Multiple colors</div>
    <div class="sixth">Transparency</div>
</body>
</html>
Output
Radial gradient
Syntax
background-image: radial-gradient(shape size at position, start-color, ..., last-color);
background-image: repeating-radial-gradient(red, yellow 10%, green 15%);
Code

<!DOCTYPE >
<html lang="en">
<head>
    <style>
        .first {
            background-image: radial-gradient(red, yellow, green);
        }
        .second {
            background-image: radial-gradient(red 5%, yellow 15%, green 60%);
        }
        .third {
            background-image: radial-gradient(circle, red, yellow, green);
        }
        .fourth {
            background-image: radial-gradient(closest-side at 60% 55%, red, yellow, grey);
        }
        .fifth {
            background-image: radial-gradient(farthest-side at 60% 55%, red, yellow, grey);
        }
        .sixth{
            background-image: radial-gradient(closest-corner, red, yellow, grey);
        }
    </style>
</head>
<body>
    <div class="first">Evenly spaced color stops(dafault)</div>
    <div class="second">Differently spaced color stops</div>
    <div class="third">Circle as shape</div>
    <div class="fourth">Closest-side</div>
    <div class="fifth">Farthest-side</div>
    <div class="sixth">Closest-corner</div>
</body>
</html>
