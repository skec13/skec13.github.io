Dropdowns
Text dropdown
Syntax
dropdown {
    position: relative;
    display: inline-block;
}

.dropdown-content {
    display: none;   /*hide element at first*/
    position: absolute;   /*to show directly beneath dropdown element*/  
}

.dropdown:hover .dropdown-content {
    display: block;   /*show content on hover*/
}
