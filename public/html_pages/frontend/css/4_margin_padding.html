Margin and padding
Margin(defines space around element)
Syntax
p {
    margin: 25px 50px 75px 100px;   /*margin: top right bottom left;*/
}
p {
    margin: 25px 50px 75px;   /*margin: top right&left bottom;*/
}
p {
    margin: 25px 50px;   /*margin: top&bottom left&right;*/
}
div {
    margin: auto;   /*will center element in another container*/
}

/*!If one element has margin-top: 50px and other margin-bot: 20px they might collapse*/
Code

<!DOCTYPE >
<html lang="en">
<head>
    <style>
        div {
            margin: auto;
            width: 200px;
            background-color: red;
            height: 50px;
        }
    </style>
</head>
    <body>
        <div></div>
    </body>
</html>
Output
Padding(defines space around elements content)
Syntax

div{
    padding: 25px 50px 75px 100px;   /*any child will have space around*/
}
div{
    width:300px;
    padding:25px;   /*PROBLEM! Because of padding <div> will have width of 350px*/
}
div{
    width: 300px;
    padding: 25px;
    box-sizing:border-box   /*SOLUTION! Width stays 300px*/
}
