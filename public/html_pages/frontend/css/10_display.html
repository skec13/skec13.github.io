Display
Type of elements
Syntax
/*Block-level Elements(start on new line and take full width available)*/
<div>, <h>, <p>, <form>, <header>, <footer>

/*Inline Elements(does not start on new line and takes up as much width as necessary)*/
<span>, <a>, <img>

p {
    display: block;   /*default for p(full width)*/
           : inline;   /*only as wide as it neeeds to be*/
           : inline-block;   /*same as inline but you can set width and height*/
           : none;   /*hides element*/
}
Code

<!DOCTYPE >
<html lang="en">
<head>
    <style>
        p {
            border: 1px solid red;
        }
        .first {
            display: inline;
        }
        .second {
            display: inline-block;
            width: 300px;
            height: 100px;
        }
    </style>
</head>
    <body>
        <p>I have display: block(default)</p>
        <p class=first>I have display: inline</p>
        <p class=second>I have display: inline-block</p>
    </body>
</html>
