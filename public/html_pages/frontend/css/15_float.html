Float
Float property
Syntax
img {
    float: right;   /*the element floats to the right of its container*/
         : left;   /*the element floats to the left of its container*/
         : none;   /*does not float, will be displayed between text*/
         : inherit;   /*inherits parents property*/
}
Code

<!DOCTYPE >
<html lang="en">
<head>
    <style>
        .first {
            float: right;
            height: 100px;
            width: 100px;
        }
        .second {
            float: left;
            height: 100px;
            width: 100px;
        }
        .third {
            float: none;
            height: 100px;
            width: 100px;
        }
    </style>
</head>
    <body>
        <b>img {float: right}</b>
        <p><img class=first src="sun.png">Lorem <p>
        <br><br><br>
        <b>img {float: left}</b>
        <p><img class=second src="sun.png">Lorem <p>
        <br><br><br>
        <b>img {float: none}</b>
        <p><img class=third src="sun.png">Lorem <p>
    </body>
</html>
Output
Clear property
Syntax
.div1 {
    float: left;
}

.div2 {
    clear: left;   /*to not show div2 in line with div1 but under*/
}
Code

<!DOCTYPE >
<html lang="en">
<head>
    <style>
        .div1 {
            float: left;
            padding: 10px;
            border: 3px solid #73AD21;
        }
        .div2 {
            padding: 10px;
            border: 3px solid red;
            float: left;
        }
        .div3 {
            float: left;
            padding: 10px;
            border: 3px solid #73AD21;
        }
        .div4 {
            padding: 10px;
            border: 3px solid red;
            clear: left;
        }
    </style>
</head>
    <body>
        <h3>Without clear</h3>
        <div class=div1>div1</div>
        <div class=div2>div2</div>
        <br><br><br><br>
        <h3>With clear</h3>
        <div class=div3>div3</div>
        <div class=div4>div4</div>
    </body>
</html>
Output
