Position
Position properties
Syntax
div{
    position: static;   /*default*/
}

div {
    position: relative;   /*relative to its normal position
                            (can set top, right, bottom and left properties)*/
    left: 30px;
}

div {
    position: absolute;   /*positioned based on nearest anchestor(element with position: relative)*/
    top: 80px;
    right: 0;
}

div {
    position: fixed;   /*positioned fixed to viewport*/
    bottom: 0;
    right: 0;
}

div {
    position: sticky;   /*based on user scroll position(toggles between relative and fixed)*/
    position: -webkit-sticky; /* Safari */
    top: 0;
}

<!DOCTYPE >
<html lang="en">
<head>
