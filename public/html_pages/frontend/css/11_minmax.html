Min/max value
min-width, max-width
Syntax
div{
    width: 500px;   /*fixed width*/
    max-width: 500px;   /*if screen smaller than 500px, element will shrink. 
                          If screen wider than 500px, element will be 500px*/
    min-width: 500px;   /*if screen smaller than 500px, elements width will stay 500px. 
                          If screen wider than 500px, element will be stretch*/                 
}
Code

<!DOCTYPE >
<html lang="en">
<head>
    <style>
        div{
            height: 100px;
            border: 1px solid blue;
        }
        .first{
            width: 500px;
        }
        .second{
            max-width: 500px;
        }
        .third{
            min-width: 500px;
        }
    </style>
</head>
    <body>
        <div class=first>width: 500px</div>
        <br>
        <div class=second>max-width: 500px</div>
        <br>
        <div class=third>min-width: 500px</div>
    </body>
</html>
