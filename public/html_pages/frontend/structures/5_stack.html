Stack
Stack implementation
Syntax
-> follows LIFO principle
-> the last element inserted into the stack is the first element to be removed
-> Push operation: adds an element to the collection
-> Pop operation: removes the most recently added element from the collection
Code

let stack = [];

stack.push(element);   //adds element to the top of the stack

stack.pop();   //removes element from the top of the stack
