Map methods
Syntax
Map.prototype.clear();   //removes all elements from this map
Map.prototype.delete(key);   //removes the specified element from this map by key
Map.prototype.entries();   //returns a new map iterator object that contains the [key, value] pairs for each element in this map
Map.prototype.forEach(callbackFn);   //executes a provided function once per each key/value pair in this map
Map.prototype.get(key);   //returns a specified element from this map
Map.prototype.has(key);   //returns a boolean indicating whether an element with the specified key exists in this map or not
Map.prototype.keys();   //returns a new map iterator object that contains the keys for each element in this map
Map.prototype.set(key, value);   //adds or updates an entry in this map with a specified key and a value
Map.prototype.values();   // returns a new map iterator object that contains the values for each element in this map
