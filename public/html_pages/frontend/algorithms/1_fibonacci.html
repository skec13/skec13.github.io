Fibonacci
Fibonacci sequence
Syntax
-> 0, 1, 1, 2, 3, 5, 8...
-> Time complexity: O(n)
Code
function fibonacci(n){
    const fib = [0, 1];
    for(let i = 2; i < n; i++){
        fib[i] = fib[i-1] + fib[i-2];
    }
    return fib;
}

Recursive Fibonacci sequence
Syntax
-> we start from the hidden-back
-> end case if F(1) = 1 and F(0) = 0
-> Time complexity: O(2^n)

function recursiveFibonacci(n){
    if(n < 2){
        return n;
    }                        
    return recursiveFibonacci(n-1) + recursiveFibonacci(n-2);
}
