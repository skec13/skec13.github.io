Climbing stairs
Climbing stairs implementation
Syntax
-> if you have n stairs how you can climb it with 1 or 2 steps at a time
-> 4 stairs (5 posibilities): (1,1,1,1), (1,1,2), (1,2,1), (2,1,1) and (2,2)

-> n = 1(1 posibility)
   n = 2(2 posibilities)
   n = 3(3 posibilities)
   n = 4(5 posibilities)
   n = 5(8 posibilities) - factorial climb of posibilities

-> Time complexity: O(n)
Code
function climbingStaircase(n){
    const noOfWays = [1,2];
    for(let i = 2; i <= n; i++){
        noOfWays[i] = noOfWays[i-1] + noOfWays[i-2];
    }
    return noOfWays[n - 1];
}
