Quick sort
Quick sort implementation
Syntax
-> pick a pivot element
   we put everything that is smaller than the pivot in left array and anything bigger in the right array
   repeat the process on smaller left and right arrays unit the arrays are the size of 1

-> Time complexity: worst O(n^2), average O(n*log(n))
Code
function quickSort(arr){
    if(arr.length < 2){
        return arr;   //base case for recursion
    }
    let pivot = arr[arr.length - 1];
    let left = [];
    let right = [];
    
    for(let i = 0; i < arr.length - 1; i++){
        if(arr[i] < pivot){
            left.push(arr[i]);
        } else{
            right.push(arr[i]);
        }
    }

    return [...quickSort(left), pivot, ...quickSort(right)]   //recursive call
}
