Power of two
Syntax
-> An integer is a power of two if there exists an integer 'x' such that n === 2^x
-> each step we divide n by 2 and see if remainder is equal to 0
-> n = 8
   8/2 = 4(remainder 0)
   4/2 = 2(remaider 0)
   2/2 = 1(remainder 0)

-> Time complexity: O(log(n))  because input size is reduced by half each iteration
Code
function isPowerOfTwo(){
    if(n < 1){
        return false;
    }                
    while(n > 1){
        if(n % 2 !== 0){
            return false;
        }
        n = n/2;
    }           
    return true; 
}

Power of two optimization
Syntax
-> in binary if number is a power of two ends with zero
-> 2 -> 10
   4 -> 100
   8 -> 1000

-> we make a bitwise and operator with the previous number 1000 & 0111 = 0000 and if we end up with zero we have a power of two
-> Time complexity: O(1)
Code
function isPowerOfTwoBitWise(n){
    if(n < 1){
        return false;
    }
    return (n & (n-1)) === 0
}
