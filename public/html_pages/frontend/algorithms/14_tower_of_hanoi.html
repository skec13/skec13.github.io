Tower of Hanoi implementation
Syntax
-> Time complexity: O(2^n - 1)  n=3->7 steps
Code
function towerOfHanoi(n, fromRod, toRod, usingRod){
    if(n === 1){
        return;   //base case, moved disk 1 from A to C
    }          

    towerOfHanoi(n-1, fromRod, usingRod, toRod);   //moving n-1 disks from A to B using C

    towerOfHanoi(n-1, usingRod, toRod, fromRod);   //moving n-1 disks from B to C using A
       
}

towerOfHanoi(3, 'A', 'C', 'B');   //shifting 3 disks from A to C with the help of B
