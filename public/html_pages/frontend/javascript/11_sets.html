Set decleration
Syntax
Set.prototype.add(value);   //adds a new value to set
Set.prototype.clear();   //removes all elements from this set
Set.prototype.delete(value);   //removes specified value from set
Set.prototype.entries();   //returns a new set iterator object that contains an array of [value, value] for each element in this set
Set.prototype.forEach(callbackFn);   //executes a provided function once for each value in this set
Set.prototype.has(value);   //returns a boolean indicating whether an element with the specified value exists in this set or not
Set.prototype.values();   //returns a new set iterator object that contains the values for each element in this set
