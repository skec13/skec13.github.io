Date formats
Syntax
const d = new Date("2015-03-25");   //ISO format(YYYY-MM-DD)
const d = new Date("2015-03");   //ISO format(YYYY-MM)
const d = new Date("2015");   //ISO format(YYYY)
const d = new Date("2015-03-25T12:00:00Z");   //ISO format(YYYY-MM-DDTHH:MM:SSZ)

const d = new Date("03/25/2015");   //Short format(MM/DD/YYYY)

const d = new Date("Mar 25 2015");   //Long format(MMM DD YYYY)

let msec = Date.parse("March 21, 2012");   //returns number between the date and January 1, 1970

Date.now();   //returns the number of millisecond elapsed since January 1, 1970

Date get methods
Syntax
Date.prototype.getFullYear();   //Get year as a four digit number (yyyy)
Date.prototype.getMonth();   //Get month as a number (0-11)
Date.prototype.getDate();   //Get day as a number (1-31)
Date.prototype.getDay();   //Get weekday as a number (0-6)
Date.prototype.getHours();   //Get hour (0-23)
Date.prototype.getMinutes();   //Get minute (0-59)
Date.prototype.getSeconds();   //Get second (0-59)
Date.prototype.getMilliseconds();   //Get millisecond (0-999)
Date.prototype.getTime();   //Get time (milliseconds since January 1, 1970)

Date set methods
Syntax
Date.prototype.setDate();   //Set the day as a number (1-31)
Date.prototype.setFullYear();   //Set the year (optionally month and day)
Date.prototype.setHours();   //Set the hour (0-23)
Date.prototype.setMilliseconds();   //Set the milliseconds (0-999)
Date.prototype.setMinutes();   //Set the minutes (0-59)
Date.prototype.setMonth();   //Set the month (0-11)
Date.prototype.setSeconds();   //Set the seconds (0-59)
Date.prototype.setTime();   //Set the time (milliseconds since January 1, 1970)

Date to string
Syntax
Date.prototype.toDateString();   //returns a string representation of the date
Date.prototype.toISOString();   //returns a string representation of the date in ISO format
Date.prototype.toString();   //returns a string representation of a date
Date.prototype.toTimeString();   //returns time of a date as a string
Code
const event = new Date(1993, 6, 28, 14, 39, 7);
console.log(event.toString());
console.log(event.toDateString());

const event1 = new Date('05 October 2011 14:48 UTC');
console.log(event1.toString());
console.log(event1.toISOString());

const event = new Date('August 19, 1975 23:15:30');
console.log(event.toTimeString());
