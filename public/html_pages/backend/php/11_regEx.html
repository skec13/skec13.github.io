Regular Expressions
Syntax
preg_match(pattern, input, matches, flags, offset);   /*returns 1 if pattern was found*/
                                                      /*pattern is a regular expression*/
                                                      /*input is a string on which the search is performed*/
                                                      /*Optional. Matches will hold array of all the matches*/
                                                      /*Optional. Flags*/
                                                      /*Optional. Offset indicates how far into the string to begin searching*/
preg_match_all(pattern, input, matches, flags, offset;   /*returns the number of times the pattern was found*/
preg_replace(patterns, replacements, input, limit, count);   /*returns a new string where matched patterns have been replaced*/
                                                             /*patterns cointains a regular expression or array of expressions*/
                                                             /*replacement string or an array of replacement strings*/
                                                             /*input is a string on which replacement is performed*/
                                                             /*Optional. Limit sets a limit to how many replacements can be made*/
                                                             /*Optional. Count will contain a number indicating how many times function ran*/
preg_grep(pattern, input, flags);   /*input in an array of strings. Function returns an array containing only elements matching a pattern*/
preg_split(pattern, string, limit, flags);   /*breaks a string into an array*/

/*pattern syntax*/
$pattern = "/\d[a-z]{1}/i"   /*\d is metacharacter*/
                             /*[a-z] used to find a range of characters*/
                             /*{1} quantifier*/
                             /*i search modifier*/

/*modifiers*/
i   performs a case-insensitive search
m   performs a multiline search
u   enables correct matching of UTF-8 encoded patterns

/*brackets*/
[abc]   find one character from the options between the brackets
[^abc]   find any character not between the brackets
[0-9]   find one character from range 0-9
[a-z]   find any character from a to z
[A-Z]   find any upper-case character from A to Z

/*quantifiers*/
n+   matches any string that contains at least one n
n*	 matches any string that contains zero or more occurrences of n
n?	 matches any string that contains zero or one occurrences of n
n{x}   matches any string that contains a sequence of X n's
n{x,y}   matches any string that contains a sequence of X to Y n's
n{x,}   matches any string that contains a sequence of at least X n's

/*metacharacters*/
|   Find a match for any one of the patterns separated by | as in: cat|dog|fish
.	Find just one instance of any character
^	Finds a match as the beginning of a string as in: ^Hello
$	Finds a match at the end of the string as in: World$
\d	 Find a digit
\s	 Find a whitespace character
\b   Find a match at the beginning of a word like this: \bWORD, or at the end of a word like this: WORD\b
\uxxxx    Find the Unicode character specified by the hexadecimal number xxxx
