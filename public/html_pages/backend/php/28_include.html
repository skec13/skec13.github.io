Autoloader
Syntax
https://www.php-fig.org/psr/psr-4/examples/
Composer(tool for dependency managment)
Syntax
composer init   //create an composer.json file in local project

composer install   //like npm-install
Composer autoloader
Syntax
vendor/composer   //when we create composer.json autoloader is created

composer dumb-autoload   //to reload autoloader

//https://www.youtube.com/watch?v=MJH1geEYMRI
Code

//composer.json
{
    "name": "matevz",
    "require": {
        mockery/mockery
    },
    "autoload": {
        "psr-4":{
            "Core\\": "Core/"   //all namespaces in Core files are named Core
        }

    }
}


//index.php
require 'vendor/autoload.php'   //include autoloader
