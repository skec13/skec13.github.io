LIKE
LIKE operator
Syntax
SELECT column1, column2, ... FROM table_name WHERE columnN LIKE pattern;
Code
//Select all customers that starts with the letter "a":
SELECT * FROM Customers WHERE CustomerName LIKE 'a%';
Output
The _ wildcard
Syntax
The _ wildcard represents a single character.
Code
/*Return all customers from a city that starts with 'L' 
followed by one wildcard character, 
then 'nd' and then two wildcard characters:*/
SELECT * FROM Customers WHERE city LIKE 'L_nd__';
Output
The % wildcard
Syntax
The % wildcard represents any number of characters, even zero characters.
Code
//Return all customers from a city that contains the letter 'L':
SELECT * FROM Customers WHERE city LIKE '%L%';
Output
Starts with
Syntax
To return records that starts with a specific letter or phrase, add the % at the end of the letter or phrase.
Code
//Return all customers that starts with 'La':
SELECT * FROM Customers WHERE CustomerName LIKE 'La%';
Output
Ends with
Syntax
To return records that ends with a specific letter or phrase, add the % at the beginning of the letter or phrase.
Code
//Return all customers that ends with 'a':
SELECT * FROM Customers WHERE CustomerName LIKE '%a';
Output
Contains
Syntax
To return records that contains a specific letter or phrase, add the % both before and after the letter or phrase.
Code
//Return all customers that contains the phrase 'or'
SELECT * FROM Customers WHERE CustomerName LIKE '%or%';
Output
Wildcards
Syntax
%	Represents zero or more characters
_	Represents a single character
[]	Represents any single character within the brackets *
^	Represents any character not in the brackets *
-	Represents any single character within the specified range *
{}	Represents any escaped character **
